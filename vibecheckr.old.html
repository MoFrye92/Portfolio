VibeCheckr Web App

A mood-based journaling and music discovery web app designed to help people check in with themselves and reset their vibe.

Overview

VibeCheckr is a personal passion project that combines journaling, music discovery and mood awareness into a simple, web-based experience. It encourages users to self‑reflect, record their feelings, and find a music vibe to match or shift their mood.

Key Features

  • Mood Check‑In: Select how you feel with an intuitive interface (emoji or word-based mood selector).
  • Journal & Reflection: Write a quick note or reflection about what’s causing your vibe.
  • Music Pairing: Get a curated playlist suggestion based on your mood via API (future integration).
  • History & Trends: See a simple trend of your moods over time (future feature).

Tech Stack

  • Languages: HTML, CSS, JavaScript
  • Frameworks/Libraries: Vanilla JS (future upgrades could include React or Vue)
  • APIs: Planned integration with Spotify or other music APIs for playlist matching.
  • Data & Storage: Currently local storage; planning to integrate a backend or cloud database for user accounts.

My Role & Lessons Learned

I designed and built VibeCheckr from scratch as a solo project. It began as a way to combine my interest in mental wellness, journaling, music, and technology. Along the way, I learned a lot about balancing simplicity with creative design and planning for features like user authentication and API integration. This project pushed me to think about user experience as much as technical implementation, and I’m excited to continue developing it.
