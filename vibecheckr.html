Overview

VibeCheckr is a mood-based journaling and music sync app designed to help users manage, explore, and shift their emotional state through self-awareness and sound. It tracks emotions and recommends playlists, journaling prompts, and mood-boosting activities based on your current state of mind.

Key Features

  • Mood Tracker: Daily check-ins with mood scales, emojis, and tags.
  • Journaling Assistant: AI-generated prompts + voice note options.
  • Music Sync: Suggests playlists based on mood (Spotify/Apple Music).
  • Emotion Analytics: Visual graphs showing emotional trends.
  • Recommendations Engine: Curated tips, affirmations, and ideas.

Tools & Technologies

  • Frontend: HTML, CSS, JavaScript (React in-progress)
  • Backend: Node.js, Express (planned)
  • APIs: Spotify API, Apple Music API
  • AI: OpenAI integration for journaling and analysis
  • Design: Figma mockups with clean, calming UI
